# Sistema de Gestão de Eventos — Arena Pernambuco (SQUAD 20)

Este projeto tem como objetivo propor um sistema para **gestão e divulgação de eventos da Arena Pernambuco**, permitindo que cidadãos acompanhem eventos, produtores proponham novas atividades e administradores gerenciem a programação do espaço.

---

Nesta primeira entrega foram desenvolvidos os seguintes artefatos do projeto:

- Histórias de usuário utilizando **BDD**
- Protótipo **Lo-Fi** no Figma
- Screencast apresentando o protótipo

## 🎨 Protótipo Lo-Fi

Protótipo de baixa fidelidade desenvolvido no **Figma**, demonstrando as principais funcionalidades do sistema.

Entre elas:

- Visualizar eventos da Arena
- Filtrar eventos
- Visualizar detalhes de eventos
- Sugerir novos eventos
- Produtor propor realização de evento
- Consultar disponibilidade de datas
- Administrador cadastrar eventos
